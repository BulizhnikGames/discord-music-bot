# Скачивание и установка

## С помощью git

```shell
git clone https://github.com/BulizhnikGames/discord-music-bot && cd discord-music-bot
go build ./cmd/main.go && ./main.exe
```

## С помощью Go

```shell
go install github.com/BulizhnikGames/discord-music-bot/cmd@latest
```
